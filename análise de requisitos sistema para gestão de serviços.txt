1. Cadastro de Clientes
Nome completo \ CPF \ Telefone \ E-mail \ CEP \ Logradouro \ Número \ complemento \ Bairro \ Cidade \ UF

2. Emissão da Ordem de Serviço (OS)
- Número da OS (gerado automaticamente)
- Data da abertura (gerado automaticamente)
- Dados do cliente (puxar do cadastro de clientes)
- Prazo estimado para conclusão
- Funcionário responsável pelo serviço
- Status da OS (Aberta, Em andamento, Finalizada, Cancelada)

3. Cadastro de Carros: 
Modelo \ Ano \ Cor \ Tipo \ Placa

4. Funcionários:
- Nome completo
- CPF  
- Telefone
- E-mail
- Cargos
- Horário de expediente
- Salario

5. Serviços:
Pendente \ Em andamento \ Concluidos \ Canceladas

6. Pagamento:
Status do pagamento \ Forma de pagamento

7. Relatório:
Histórico de serviços
Total faturado.
